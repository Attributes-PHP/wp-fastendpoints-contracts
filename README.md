# WP-FastEndpoints Contracts

Holds common contracts for **[WP-FastEndpoints](https://github.com/matapatos/wp-fastendpoints)**.

## Requirements

- PHP 8.1+

## Installation

```bash
composer require attributes-php/wp-fastendpoints-contracts
```
